# Homelab 

## Introduction

Hello everyone! My name is Flavio, I'm a Computer Science Student interested in Cyber Security. This homelab is a personal project created to develop and improve my skills/knowledge in cybersecurity. My goal is to build a realistic environment that mirros how modern corporate networks are structured and secured.

Using virtualization, the idea is not only to study theory, but to apply it in a controlled environment where I can experiment, break things, fix them, and truly understand how systems behave in real-world scenarios. This repository also serves as documentation and reference material, enabling others to replicate similar environments and use it as a learning resource for cybersecurity, networking, and infrastructure studies.

This repository includes a technical journal where I document major updates, architectural decisions, experiments, and lessons learned throughout the development of this homelab.
The writing style is intentionally personal, but technical. My goal is not only to describe configurations and commands, but to explain the reasoning behind decisions, the challenges encountered, and the security implications involved in each change. This journal reflects my learning process — including mistakes, troubleshooting steps, and improvements over time. By documenting everything in a structured way, I aim to consolidate my own knowledge while also creating a resource that others can use to build and understand their own cybersecurity-focused environments.
